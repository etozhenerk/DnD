# Библиотека визуальных концептов

Версионируемый набор утверждённых и безопасных для повторного использования изображений. Эта папка — визуальный источник истины для персонажей, рас, мира и UI-стиля.

## Статусы

- `canonical` — утверждённое изображение конкретной сущности; можно показывать в приложении.
- `reference` — ориентир стиля или расы; использовать при генерации, но не выдавать за отдельную сущность.

Статусы и связи перечислены в `manifest.json`.

## Структура

```text
characters/  Канонические портреты героев
races/       Визуальные ориентиры рас
style/       Общая композиция команды и макет карточки
world/       Карта, визуальная география и прозрачные PNG регионов
manifest.json
```

## Общая визуальная формула

Персонажи выполнены как кинематографичное полуреалистичное фэнтези: узнаваемые лица реальных людей, полный рост, натуральные материалы костюма, холодная сине-белая среда и мягкое рассеянное освещение. Расовые признаки должны дополнять лицо, а не уничтожать сходство.

Новые концепты добавлять сюда только после одобрения пользователя. Исходные фотографии реальных людей сюда не копировать.

PNG в `world/regions/` вырезаны по естественному силуэту суши, облаков, крон и водопадов. Полигоны из `content/world-map.json` независимо описывают кликабельные зоны по пользовательской разметке. Красная линия и окружающее море не являются частью итоговых ассетов.
