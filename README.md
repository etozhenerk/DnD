# D&D Campaign Service

Кликабельный сервис для проведения авторских D&D-кампаний на одном общем экране. Главная карта делится на области; каждая область открывает самостоятельную кампанию с локациями, персонажами, встречами и сценарием для мастера.

## Структура

```text
content/                 Канонические игровые данные
  characters.json       Герои
  races.json            Расы
  rules.json            Домашняя d20-система
  world-map.json        Кликабельные регионы карты
  campaigns/            Кампании и мастерские сценарии
skills/                  Локальные Codex-скиллы проекта
assets/concepts/         Версионируемые визуальные концепты
AGENTS.md                Правила работы агентов
```

## Визуальные концепты

Папка `assets/concepts/` содержит оптимизированные утверждённые портреты героев, референсы рас, карту и стиль карточек. Её файлы входят в Git и являются стабильным входом для скиллов и будущего фронтенда. Полный индекс находится в `assets/concepts/manifest.json`.

## Локальные скиллы

- `$maintain-dnd-world` — изменение схемы и поддержка связности данных.
- `$create-dnd-character` — создание нового героя.
- `$create-dnd-enemy` — создание врага, злодея или боевой группы.
- `$create-dnd-campaign` — создание кампании для новой области карты.

Скиллы лежат в `skills/` и маршрутизируются через `AGENTS.md`.
