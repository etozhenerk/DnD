# Контракт локальной сессии

## Снимок

Сессия хранит версию схемы, campaign id, время обновления, активную сцену, участников, инициативу, номер раунда, текущий ход, инвентарь, флаги, счётчики, отношения, улики, состояния локаций, модификаторы финала и журнал событий.

Участник хранит ссылку на канонический `id` и сессионные HP, max HP, AC, атаку, характеристики, uses, charges, conditions и временные модификаторы.

## Поток изменения

```text
command -> validation -> events -> reducer -> snapshot -> localStorage
```

React-компонент не изменяет снимок напрямую.

## Коррекция

Отмена создаёт `action-corrected` со ссылкой на исправляемое событие и компенсирующими событиями. История не удаляется.

## Восстановление

При загрузке проверить version, campaign id и целостность журнала. Повторное применение одного command id не должно менять состояние дважды.
